# Mobile and tablet responsive layout

Version v46 keeps the same public URLs and same HTML files, but adds responsive CSS and a JavaScript mobile navigation toggle.

Desktop browsers use the full header navigation. Tablet and phone widths show a compact header with a Menu button at the upper right. The menu expands to show Browse, Submission, Guidelines, Topics, Policies, About, and Login / Member dashboard.

The project intentionally does not include wrangler.toml. Cloudflare Pages settings, D1 bindings, R2 bindings, and secrets should be managed from the Cloudflare Dashboard.
