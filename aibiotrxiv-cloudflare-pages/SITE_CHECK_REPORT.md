# AIBioTrXiv v42 site check

## Scope
This build extends v41 with responsive phone and tablet support while keeping the same Cloudflare Pages project structure.

## Changes
- Added a mobile/tablet navigation button generated by `app.js` on every page using the existing header.
- Desktop navigation remains unchanged above 1100px.
- Tablet and phone layouts collapse grids, article pages, forms, admin tables, member dashboard cards, PDF workspace, and footer into mobile-friendly layouts.
- Header height is reduced on phone/tablet.
- Mobile navigation includes Browse, Submission, Guidelines, Topics, Policies, About, and Login.
- Article anchor offsets were reduced for the smaller mobile header.

## Deployment structure
The ZIP root contains `aibiotrxiv-cloudflare-pages/`, and website files are inside that folder.
